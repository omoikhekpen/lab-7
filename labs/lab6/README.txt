Lab 6 - Efe's Cakes Portfolio
Efe Ivagba (CS ID: ivagba, awubaefe@dal.ca)

Description:
This React-based portfolio site serves as a creative bakery website called "Efe's Cakes." It showcases our delectable cake offerings,
our story, and our passion for baking. The site features a Home page, an About page, a Products page (displaying our cake menu), and a 404 page.
It uses React Router for navigation, Bootstrap 5 for styling, and is designed with accessibility in mind following WCAG guidelines.

Built With:
- React (https://reactjs.org/)
- React Router (https://reactrouter.com/)
- Bootstrap 5 (https://getbootstrap.com/)
- Create React App (or Vite) for project setup
